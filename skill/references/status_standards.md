# 病历管家 - 异常状态统一标准规范

> 版本: v1.0 | 生效日期: 2026.04.07

---

## 一、状态图标体系 / Status Icon System

### 1.1 基础状态图标 (4级)

| 图标 | 名称 | 颜色代码 | 含义 | 适用场景 |
|------|------|----------|------|----------|
| ✅ | 正常 | 绿色 | 指标在参考范围内 | 所有报告类型 |
| ⚠️ | 轻度异常 | 黄色/橙色 | I-II级异常或需关注 | 轻度偏离正常 |
| 🔴 | 重度异常 | 红色 | III-IV级或危急值 | 严重异常 |
| ➖ | 未检测/无数据 | 灰色 | 该次检查未包含此指标 | 可选填充 |

### 1.2 趋势箭头 (用于数值后缀)

| 箭头 | 含义 | 使用条件 |
|------|------|----------|
| ↑ | 高于正常上限 | 数值 > 参考范围最大值 |
| ↓ | 低于正常下限 | 数值 < 参考范围最小值 |
| → | 在正常范围内 | 数值在参考范围内（可选显示） |
| ↗ | 较上次升高 | 与历史数据对比时使用 |
| ↘ | 较上次降低 | 与历史数据对比时使用 |

## 二、血常规指标状态标准 / Blood Test Standards

### 2.1 分级与图标映射

| 异常级别 | 图标 | 显示格式 | 示例 |
|----------|------|----------|------|
| **0级 正常** | ✅ 或 无标记 | `数值` 或 `✅数值` | `7.22` 或 `✅7.22` |
| **I级 轻度** | ⚠️ | `⚠️数值↓` | `⚠️0.61↓` |
| **II级 中度** | 🟠 | `🟠数值↓↓` | `🟠1.36↓↓` |
| **III级 重度** | 🔴 | `🔴数值↓↓↓` | `🔴0.42↓↓↓` |
| **IV级 危急** | 🚨 | `🚨数值↓↓↓↓` | `🚨0.08↓↓↓↓` |

### 2.2 特殊阈值警示

| 指标 | 危急值阈值 | 特殊标识 |
|------|-----------|----------|
| ANC | < 0.5×10⁹/L | 自动添加 `⚠️感染风险` |
| PLT | < 20×10⁹/L | 自动添加 `🚨出血危象` |
| Hb | < 60 g/L | 自动添加 `🚨重度贫血` |
| CRP | > 100 mg/L | 自动添加 `🔴严重感染可能` |
| 体温+低ANC | 任何发热 | `🚨急诊` |

## 三、血脂指标状态标准 / Lipid Standards

### 3.1 分级与图标映射

| 状态 | 图标 | 显示格式 | 说明 |
|------|------|----------|------|
| **正常范围** | ✅ | `✅数值` 或仅 `数值` | TC<5.2, TG<1.69等 |
| **边缘升高** | ⚠️ | `⚠️数值↑` | 轻微超出参考范围 |
| **升高/异常** | 🟠 | `🟠数值↑↑` | 明显异常，需干预 |

## 四、肿瘤标志物状态标准 / Tumor Markers Standards

### 4.1 分级与图标映射

| 状态 | 图标 | 显示格式 | 说明 |
|------|------|----------|------|
| **正常范围** | ✅ | `✅数值` 或仅 `数值` | 指标在参考范围内 |
| **轻度升高** | ⚠️ | `⚠️数值↑` | 轻微超出，需关注趋势 |
| **明显升高** | 🔴 | `🔴数值↑↑` | 明显异常，需关注病情 |

### 4.2 常用指标参考值

| 指标 | 参考范围 | 临床意义 |
|------|---------|----------|
| CA-125 | 0－35 U/mL | 卵巢癌/肺癌标志物 |
| CYFRA21-1 | 0-3.3 ng/mL | 鳞癌标志物 |
| VEGF | 0-160 pg/mL | 血管生成指标 |
| SCC | 0-3 ng/ml | 鳞状细胞癌抗原 |
| CEA | 0-5 ng/ml | 广谱肿瘤标志物 |
| CA19-9 | 0-37 U/mL | 消化道/胰腺癌标志物 |

### 4.3 趋势判断

| 变化模式 | 图标 | 含义 |
|----------|------|------|
| 持续下降或稳定在正常范围 | ✅ | 治疗有效，继续当前方案 |
| 波动但总体稳定 | ⚠️ | 定期监测，关注变化趋势 |
| 持续上升 | 🔴 | 可能进展，建议就医评估 |

## 五、CT影像报告状态标准 / CT Imaging Standards

### 4.1 总体评价图标

| 评价级别 | 图标 | 关键词示例 |
|----------|------|-----------|
| **稳定/好转** | ✅ | 稳定、好转、缩小、吸收、消失 |
| **稳定但需关注** | ⚠️ | 稳定但有新发病变、炎症增多 |
| **进展/恶化** | 🔴 | 进展、增大、增多、转移、新发灶 |

### 4.2 局部病灶描述格式

```
[图标] 描述内容
例: 🔴右肺下叶结节增大至1.2cm(原0.8cm)
    ✅左肺原发灶同前相仿
    ⚠️纵隔淋巴结较前稍增多
```

## 五、异常汇总列标准 / Abnormal Summary Format

### 5.1 格式规则

**有异常时：**
```
⚠️指标1变化,指标2变化,指标3变化
例: ⚠️白细胞↓,中性粒↓,CRP↑
```

**全部正常时：**
```
✅各项指标均在正常范围内
```

**CT报告特殊格式：**
```
✅整体病情稳定可控
⚠️病变整体稳定，右肺炎症性改变增多
🔴病变进展，出现新发转移灶
```

### 6.1 图标与单元格背景色对应

## 七、实施要求 / Implementation Requirements

### 7.1 新数据录入规范

1. **必须使用本标准的图标和格式**
2. **箭头数量表示异常程度**: 
   - 轻度: 1个箭头 (↑/↓)
   - 中度: 2个箭头 (↑↑/↓↓)
   - 重度: 3个箭头 (↑↑↑/↓↓↓)
   - 极重: 4个箭头 + 🚨标识

### 7.2 历史数据处理

- 不强制回溯修改已有数据
- 新录入时可按新标准格式化
- 优先保证新增数据一致性

### 7.3 SKILL.md 引用

本标准作为 `medical-record-butler/SKILL.md` 的附件执行，所有自动录入和人工审核均遵循此标准。
