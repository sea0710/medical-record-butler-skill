## ✨ 核心特性

- 📸 **智能识别**: 自动识别血常规检查报告图片
- 🔍 **异常判断**: 基于0-IV级分级评估
- 📊 **趋势分析**: 自动生成历史对比图表
- 📝 **自动记录**: 自动更新到飞书文档,无需手动操作
- 💡 **专业建议**: 提供处理意见和自我监测指导

## 🚀 快速开始

### 1. 安装Skill

下载最新的 `chemo-blood-monitor.zip` 文件,在CodeBuddy中安装。

### 2. 配置飞书文档

按照 `docs/飞书配置指南.md` 完成飞书开放平台配置。

### 3. 开始使用

直接发送血常规检查报告图片,AI会自动:
- 识别并分析报告
- 判断异常指标
- 自动更新到您的飞书文档
- 提供处理建议
