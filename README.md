### ✨ 核心功能特性
#### 📸 多类型报告智能识别
- **全场景覆盖**：已适配**血常规、血脂监测、肿瘤标志物、CT/MRI影像报告**四大类医疗报告（v2.0重大升级）
- **深度提取**：精准识别核心指标数值、单位、参考范围及异常标记
- **格式统一**：自动标准化数据格式，解决不同医院报告样式不统一的痛点

#### 🔬 指标自动提取与格式化
- OCR智能解析报告图片，提取关键指标（WBC/ANC/PLT/Hb/CRP/LYM#等）
- 自动统一单位换算与数值格式化，消除"120"与"120.00"等歧义
- 自动区分性别阈值（如Hb血红蛋白：男≥120 / 女≥110）

#### ⚠️ 异常状态分级评估
基于权威临床参考值，建立**0-IV级异常分级体系**，自动图标化展示：
- **0级** (无标记)：正常范围 → 无需处理
- **I级** (⚠️)：轻度异常 → 观察监测
- **II级** (🟠)：中度异常 → 建议就诊
- **III级** (🔴)：重度异常 → 紧急就医
- **IV级** (🚨)：极重度/危急值 → **立即就医**

#### 💊 智能处理建议生成
结合出院医嘱与临床指南，自动生成分级处理原则：
- 自我监测指导（症状观察要点）
- 生活护理建议（饮食/活动注意）
- 危急值紧急处理预案（如ANC < 0.5×10⁹/L需立即联系医生）

#### 📊 台账自动规整与可视化
- **自动建表**：自动在飞书表格中创建规范工作表（01-血脂/02-血常规/03-CT影像等）
- **历史归档**：自动录入飞书表格，形成完整治疗数据台账
- **趋势对比**：支持历史数据趋势可视化，直观评估治疗效果

### 🚀 快速开始
#### 方式一：直接安装（推荐）
1. 下载 `skill` 目录到本地
2. 将整个 `skill` 文件夹放入 CodeBuddy 的 `skills` 目录
3. 重启 CodeBuddy 即可使用

#### 方式二：飞书表格集成（可选）
如需自动录入飞书表格功能：
1. 配置 `sharing_guide.md` 中的飞书开发者权限
2. 运行 `feishu_api.py` 脚本完成授权
3. 自动同步数据至飞书多维表格
