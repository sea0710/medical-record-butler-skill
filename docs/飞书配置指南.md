# 飞书配置指南

## 📋 准备工作

在开始配置前,您需要:

- ✅ 飞书账号
- ✅ 手机或电脑
- ✅ 10-15分钟时间

---

## 🚀 配置步骤

### 步骤1: 注册飞书开放平台

1. 访问 https://open.feishu.cn/
2. 点击右上角"登录"
3. 使用您的飞书账号登录
4. 进入"开发者后台"

---

### 步骤2: 创建企业自建应用

1. 点击"创建企业自建应用"
2. 填写应用信息:
   ```
   应用名称: 化疗血常规监测助手
   应用描述: 自动记录化疗血常规检查结果
   应用图标: 选择一个合适的图标
   ```
3. 点击"创建"

---

### 步骤3: 获取应用凭证

创建成功后:

1. 进入应用详情页
2. 点击左侧菜单"凭证与基础信息"
3. 复制以下信息并保存:
   - **App ID**: `cli_xxxxxxxxxxxxxx`
   - **App Secret**: `xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

⚠️ **重要**: App Secret是敏感信息,请妥善保管!

---

### 步骤4: 配置权限

1. 点击左侧菜单"权限管理"
2. 点击"申请权限"
3. 搜索并添加以下权限:

#### 必需权限:
```
✅ docx:document - 查看、评论、编辑和管理云空间中所有云文档
✅ docx:document:readonly - 查看云空间中所有云文档
```

#### 添加方法:
1. 在搜索框输入权限名称
2. 点击"申请权限"
3. 选择"所有员工可用"
4. 点击"确定"

---

### 步骤5: 创建飞书文档

1. 打开飞书App或网页版
2. 点击"云文档"
3. 点击"新建" → "文档"
4. 输入标题: "化疗血常规监测记录"
5. 保存文档
6. 从浏览器地址栏复制文档ID

**文档链接格式**:
```
https://feishu.cn/docx/[文档ID]
                    ↑↑↑↑↑↑↑↑↑↑
                    这部分就是文档ID
```

**示例**:
```
完整链接: https://feishu.cn/docx/TDEKdMYWmoykbCx2jj7cym8vnqf
文档ID: TDEKdMYWmoykbCx2jj7cym8vnqf
```

---

### 步骤6: 发布应用

1. 回到开发者后台
2. 点击"版本管理与发布"
3. 点击"创建版本"
4. 填写版本信息:
   ```
   版本号: 1.0.0
   更新说明: 初始版本
   ```
5. 点击"保存"
6. 点击"申请上线发布"
7. 等待审核通过(通常几分钟)

---

### 步骤7: 配置Skill

将获取的信息提供给AI:

```
飞书应用配置信息:
- App ID: cli_xxxxxxxxxxxxxx
- App Secret: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
- 文档ID: TDEKdMYWmoykbCx2jj7cym8vnqf
```

AI会自动:
1. 创建配置文件
2. 测试连接
3. 验证权限
4. 完成集成

---

## ✅ 验证配置

配置完成后,AI会进行测试:

1. **认证测试**: 验证App ID和App Secret
2. **权限测试**: 验证文档访问权限
3. **写入测试**: 尝试写入测试数据

全部通过后,配置完成! ✅

---

## 🔧 常见问题

### Q1: 提示"权限不足"怎么办?
A: 确保已添加所有必需权限,并等待应用发布成功。

### Q2: 提示"文档不存在"怎么办?
A: 检查文档ID是否正确,确保文档未被删除。

### Q3: App Secret忘记了怎么办?
A: 在开发者后台可以重置App Secret。

### Q4: 需要付费吗?
A: 飞书开放平台基础功能免费,完全够用。

### Q5: 可以用个人账号吗?
A: 可以!个人账号也能创建企业自建应用。

## 🔒 安全说明

### 数据安全
- 所有数据存储在您的飞书账号中
- 由飞书平台保障安全
- AI只负责分析和记录

### 隐私保护
- App Secret仅用于API认证
- 不会泄露给第三方
- 可随时重置

### 权限控制
- 只访问您授权的文档
- 不会访问其他云文档
- 可随时撤销权限
