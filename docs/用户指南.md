# 用户指南 - 化疗血常规监测Skill

## 📋 准备工作

### 1. 安装CodeBuddy
确保您已安装CodeBuddy AI助手。

### 2. 安装Skill
下载 `chemo-blood-monitor.zip` 并在CodeBuddy中安装。

### 3. 配置飞书文档
按照《飞书配置指南.md》完成飞书开放平台配置。

## 🚀 使用流程

### 第一步：配置飞书文档

1. 创建飞书开放平台账号
2. 创建企业自建应用
3. 获取App ID和App Secret
4. 配置API权限
5. 创建飞书文档

详细步骤请参考《飞书配置指南.md》

### 第二步：发送检查报告

直接拍摄或扫描血常规检查报告,发送图片给AI:

```
方式1: 直接发送图片
[上传图片] → AI自动识别

方式2: 带问题描述
"请帮我分析这张血常规报告"
[上传图片]
```

### 第三步：查看结果

AI会自动完成以下操作:

1. **识别报告** - 提取所有指标数据
2. **分析异常** - 判断异常级别(0-IV级)
3. **生成报告** - 详细的分析和建议
4. **更新文档** - 自动更新到您的飞书文档
5. **提供链接** - 给您文档链接查看完整记录

## 📊 功能详解

### 1. 核心指标监测

系统重点关注以下指标:

| 指标 | 英文 | 单位 | 临床意义 |
|------|------|------|---------|
| 白细胞 | WBC | ×10⁹/L | 免疫功能 |
| 中性粒细胞 | ANC | ×10⁹/L | 感染风险 |
| 血小板 | PLT | ×10⁹/L | 出血风险 |
| 血红蛋白 | Hb | g/L | 贫血评估 |

### 2. 异常分级标准

| 级别 | 严重程度 | 处理建议 |
|------|---------|---------|
| 0级 | 正常 | 继续监测 |
| I级 | 轻度异常 | 密切观察 |
| II级 | 中度异常 | 医学咨询 |
| III级 | 重度异常 | 及时就医 |
| IV级 | 严重异常 | 立即就医 |

### 3. 历史趋势分析

系统会自动生成:

- 📈 **趋势对比表** - 所有检查在同一表格对比
- 📊 **可视化图表** - 柱状图显示指标变化
- 📉 **变化幅度** - 百分比量化改善或恶化
- 💡 **趋势判断** - 清晰标注改善/稳定/下降

## 💡 使用技巧

### 拍照建议

✅ **推荐做法**:
- 光线充足,避免阴影
- 报告平整,无折叠
- 字迹清晰可见
- 包含完整的指标数据

❌ **避免**:
- 光线暗淡
- 报告模糊
- 字迹不清
- 数据缺失

### 最佳实践

1. **定期检查** - 按医嘱定期复查
2. **及时上传** - 检查后尽快发送报告
3. **关注趋势** - 重点查看历史对比
4. **遵医嘱** - AI建议仅供参考

## 🔔 紧急情况判断

### 立即就医的情况

⚠️ 以下情况需要立即就医:

- ANC < 0.5×10⁹/L (严重中性粒细胞减少)
- PLT < 20×10⁹/L (出血危机)
- Hb < 60 g/L (严重贫血)
- 体温 ≥ 38.0℃ (发热)
- 活动性出血
- 严重症状(胸痛、呼吸困难等)

## ❓ 常见问题

### Q1: 飞书文档链接是什么?
A: 系统会自动创建并维护您的专属飞书文档,每次检查都会自动更新到这个文档。

### Q2: 如何查看历史记录?
A: 打开您的飞书文档链接,所有检查记录都会保存在那里,最新的在最上面。

### Q3: 数据安全吗?
A: 所有数据存储在您的飞书账号中,由飞书平台保障安全,AI只负责分析和记录。

### Q4: 可以多人使用吗?
A: 可以!每个用户配置自己的飞书应用,拥有独立的文档空间。

### Q5: 如何获取更新?
A: 定期检查GitHub仓库的releases页面,下载最新版本重新安装即可。
